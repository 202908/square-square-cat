# 貓貓方塊疊疊樂遊戲

一個 Node.js + WebSocket + Three.js 的多人連線網頁遊戲原型。玩家可以登入帳號、用訪客身份進入、在 3D 星球島上看到其他在線玩家、聊天、疊疊樂、攻擊、兌換金幣代碼、逛商城和裝備背包物品。

## 目前功能

- 開始畫面與登入流程
- 創建新帳號：1 到 10 個英文字或數字
- 登入已有帳號：會讀取之前保存的進度
- 訪客登入：直接從等級 1 開始，不保存進度
- 主機帳號：由本機或部署平台的 `HOST_ACCOUNT_CODE` 環境變數設定，不要寫進 GitHub
- 主機可以在遊戲內新增或下架兌換代碼；不要把真實代碼寫進公開檔案
- 地圖上會出現可收集的黃色金幣，玩家碰到後會獲得 1 金幣
- 多人 WebSocket 同步
- 3D 方塊貓、星球島、遊樂設施雛形
- 電腦鍵盤控制與 iPad 觸控搖桿
- 聊天與語音輸入按鈕
- 商城、背包、好友功能骨架

## 本機啟動

```bash
npm install
npm start
```

打開：

```text
http://localhost:3000
```

可以開兩個瀏覽器視窗測試多人同步。

如果你要啟用主機帳號，請在啟動 server 時設定：

```bash
HOST_ACCOUNT_CODE="你的主機登入碼" HOST_PASSWORD="你的主機密碼" npm start
```

不要把真正的主機登入碼或主機密碼寫進 README、程式碼或公開聊天。

## 操作

- 電腦移動：方向鍵或 WASD
- 電腦跳躍：空白鍵
- 疊疊樂：畫面按鈕或 `E`
- 攻擊：畫面按鈕或 `M`
- iPad/觸控：左下搖桿移動，右下按鈕跳躍、疊疊樂、攻擊
- 裝備翅膀後：會出現上/下飛行按鈕

## 上傳 GitHub 與上線

這個專案可以直接放到 GitHub。因為它有 WebSocket server，所以不能只靠 GitHub Pages 完整運作。GitHub Pages 只能放純前端，這個遊戲上線時需要能跑 Node.js 的平台，例如 Render、Railway、Fly.io 或 VPS。
